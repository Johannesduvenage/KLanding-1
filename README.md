## Hot to start
1. Install node ([How to] (https://nodejs.org/en/download/package-manager/)).
2. Clone https://github.com/kiskinvlad/KLanding-1
3. npm i
4. grunt server
5. Go to localhost:3000
6. For Dev - Install sass ([How to] (http://sass-lang.com/install)).
